# maxrichie.ai

Terminal-style landing page for **Maxrichie** — a Singapore-based AI startup.

Static site. No build step. Just open `index.html`.

## Local preview

```bash
python3 -m http.server 8080
# then visit http://localhost:8080
```

## Deploy

Hosted via **GitHub Pages** from the `main` branch root.
Settings → Pages → Source: `main` / root.

## Stack

- Vanilla HTML / CSS / JS
- JetBrains Mono (Google Fonts)
- Canvas matrix rain
- CRT scanline + vignette overlays

## Commands available in the terminal

`help`, `about`, `mission`, `team`, `stack`, `contact`, `ls`, `cat <section>`,
`whoami`, `date`, `echo`, `ai <msg>`, `theme <green|amber|blue>`, `matrix`,
`banner`, `clear`, `sudo`.
